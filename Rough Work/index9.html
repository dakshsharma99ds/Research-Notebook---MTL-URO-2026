Chemical Reaction Network Simulation – Boekhoven Lab

Tools: Python, Numpy, Matplotlib

Objective: Explore dynamics of a simple chemical network (A → B → C).

Show/Hide Code
import numpy as np
import matplotlib.pyplot as plt

t = np.linspace(0, 10, 100)
A = np.exp(-0.5*t)
B = 0.7*(1 - np.exp(-0.3*t))
C = 1 - A - B

plt.plot(t,A,label='A')
plt.plot(t,B,label='B')
plt.plot(t,C,label='C')
plt.xlabel('Time'); plt.ylabel('Concentration')
plt.title('Chemical Reaction Network Simulation')
plt.legend(); plt.show()
Observation: Increasing the reaction rate of A → B causes B to peak sooner and C forms earlier. Demonstrates chemical network dynamics.
Future Work: Test different network topologies and rate constants to see emergent behaviors.

Self-Organization Simulation – Sarah Loss Lab / MPI-DS

Tools: Python, Matplotlib, py-pde

Objective: Visualize how simple local rules generate complex patterns in a 2D system.

Show/Hide Code
import numpy as np
import matplotlib.pyplot as plt
from pde import ScalarField, ReactionDiffusionPDE, UnitGrid

grid = UnitGrid([50,50], periodic=True)
state = ScalarField(grid, np.random.rand(grid.size))
eq = ReactionDiffusionPDE(a=0.1, b=0.05, D=0.1)

snapshots = []
current_state = state.copy()
for i in range(0,51,10):
    result = eq.solve(current_state, t_range=10, dt=0.1, tracker=None)
    snapshots.append(result.data.copy())
    current_state = result

plt.imshow(snapshots[-1].reshape(grid.shape), origin='lower', cmap='plasma')
plt.show()
Observation: Random fluctuations evolve into distinct patterns over time, showing how local rules create global structure.
Future Work: Explore parameter changes and initial conditions to study emergent pattern diversity.

1D Diffusion – Zwicker Lab

Tools: Python, Matplotlib, py-pde

Objective: Simulate how a concentration spike spreads over time in 1D.

Show/Hide Code
import numpy as np
import matplotlib.pyplot as plt
from pde import ScalarField, DiffusionPDE, UnitGrid

grid = UnitGrid([100], periodic=False)
x = grid.cell_coords[:,0]
state = ScalarField(grid, np.exp(-100*(x-0.5)**2))
eq = DiffusionPDE(diffusivity=0.01)
snapshots = []
current_state = state.copy()
for i in range(0,51,10):
    result = eq.solve(current_state, t_range=10, dt=0.1, tracker=None)
    snapshots.append(result.data.copy())
    current_state = result

plt.plot(x,snapshots[-1])
plt.show()
Observation: Spike spreads gradually along 1D line, demonstrating diffusion dynamics.
Future Work: Explore effects of boundary conditions and diffusion constants.

2D Diffusion – Zwicker Lab

Tools: Python, Matplotlib, py-pde

Objective: Simulate concentration spreading across a 2D grid.

Show/Hide Code
import matplotlib.pyplot as plt
from pde import ScalarField, DiffusionPDE, UnitGrid

grid = UnitGrid([10,10], periodic=False)
state = ScalarField(grid); state.data[:] = 0
center_index = len(state.data)//2
state.data[center_index] = 1
eq = DiffusionPDE(diffusivity=0.1)
result = eq.solve(state, t_range=5, dt=0.1)
plt.imshow(result.data.reshape(grid.shape), origin='lower', cmap='viridis'); plt.show()
Observation: Spike spreads outward smoothly, visualizing 2D diffusion dynamics.
Future Work: Test different grid sizes and diffusivities for pattern evolution.
